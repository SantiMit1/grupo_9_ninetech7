
# Ninetech Ecommerce

Ecommerce destinado a la venta de hardware y perifericos de computadora
## Features

- Ecommerce funcional
- Registro y Login de usuarios
- CRUD de productos
- Funciones extras para cuentas con permisos de administraci√≥n


## Tech Stack

**FrontEnd:** HTML, CSS, Javascript

**BackEnd:** Node, Express


## Ejecutar localmente

Clone el proyecto

```bash
  git clone https://github.com/SantiMit1/ninetech.git
```

Vaya al directorio del proyecto

```bash
  cd ninetech
```

Instale las dependencias

```bash
  npm install
```

Ejecute el script "database.sql" ubicado en la carpeta "database files"

Inicie el servidor

```bash
  npm run start
```
